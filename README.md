## RPS-Game

## About the game

The game is to bring your childhood memories back.
you can enjoy with  the  bot.
 Rock wins against Scissors.
 Scissors wins against Paper.
 Paper wins against Rock..
 If both player shows the same signal it will tie.
 HAVE A FUN!...
